# 🦖 Dino Run Game

A simple recreation of the classic Google Chrome Dino Run game, built with Python and Pygame.

## 🌟 About The Project

This project is a fun, introductory game developed to practice basic game development concepts. The player controls a dinosaur that must avoid obstacles (cacti and pterodactyls) by jumping and ducking. The game's speed increases over time, making it progressively more challenging.

## ✨ Features

- Endless running and obstacle generation
- Jumping and ducking mechanics
- Scoring system
- Increasing difficulty over time
- Sound effects for jumping, scoring, and game over

## 💻 Technologies Used

- **Python**: The core programming language.
- **Pygame**: A set of Python modules designed for writing video games.

## 🚀 How to Run

1.  **Prerequisites:**
    - Make sure you have Python installed on your system.

2.  **Clone the repository (or download the files):**
    ```sh
    git clone <your-repository-url>
    cd Dino-Run-main
    ```

3.  **Install dependencies:**
    This project uses `pygame`. You can install it using pip:
    ```sh
    pip install pygame
    ```

4.  **Run the game:**
    Execute the `main.py` file to start the game.
    ```sh
    python main.py
    ```

## 🎮 Controls

- **Spacebar / Up Arrow**: Jump
- **Down Arrow**: Duck
- **P / Esc**: Pause game
